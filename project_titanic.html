Titanic Survival Prediction

A machine learning project to predict the survival of passengers on the Titanic.

Project Description

This project involves building a machine learning model to predict whether a passenger survived the Titanic disaster based on various features

Key Features

  • Data Cleaning and Preprocessing
  • Exploratory Data Analysis (EDA)
  • Feature Engineering
  • Model Selection and Evaluation

Technologies Used

  • Python
  • NumPy
  • Pandas
  • Scikit-learn
  • Matplotlib & Seaborn

Project Links

  • GitHub Repository
  • Live Demo

Challenges Faced

Handling missing data, selecting relevant features, and tuning model hyperparameters to improve accuracy.
